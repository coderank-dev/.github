# CodeRank

AI-optimized library documentation for coding agents. 5–10× fewer tokens.

- 🔧 **CLI-first** — zero MCP overhead, works everywhere
- 📦 **Modular markdown** — API surface + topic files, not flat blobs
- ⚡ **<100ms** — Cloudflare edge, no round trips
- 🔒 **Offline + self-hosted** — local cache, enterprise deployment
- 🎯 **Stack-aware** — `.coderank.yml` configures preferred libraries

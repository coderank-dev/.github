# Contributing to CodeRank

## Development Setup

1. Install Go 1.22+, Node.js 18+
2. Clone the repo you want to work on
3. Follow the repo-specific README

## Commit Convention

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `test:` tests
- `refactor:` code refactoring
- `chore:` maintenance

## Pull Requests

- One feature per PR
- Include tests
- Update relevant docs
